GameMaster Color Chat V1.0

Description: a small patch to make a color chat for gamemasters (works on all chats say, whisper etc.).

NOTE: check the config settings and make sure you have ingame ".gm chat on"

How to add: safe all as "gm_color_chat.patch" file add with git bash "patch -p1 > gm_color_chat.patch" compile and have fun.
